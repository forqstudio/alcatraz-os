# AI User Development Environment

This directory contains the configuration for the `al` user, designed to run [OpenCode](https://opencode.ai) - an AI coding agent.

## Tools

| OpenCode Tool | Alternative | Why it's better |
|---------------|-------------|-----------------|
| Glob | fd | Faster, parallel execution |
| Grep | ripgrep | 10x+ faster, better defaults |
| Read | bat | Syntax highlighting, line numbers |
| - | tree | Visual directory structure |
| - | git | Version control |
